

    ##项目:LacRadio
    ###项目功能介绍
    - LacRadio是一个在线音乐播放器,提供多个专辑选择.
    - 支持响应式大屏适配    vh 媒体查询
    - 实现基本交互:播放暂停,专辑切换
    - 有华丽的音乐播放页面,特效歌词展示,播放进度条
        
    ###项目技术细节介绍

    - 使用媒体查询,配合vh实现响应式大屏适配
    - 使用jQuery,ajax实现页面后台数据处理
    - 使用事件监听完成用户ui交互
